TakeNote

List-based JavaScript WYSIWYG Editor

TakeNote is a simple WYSIWYG editor written in CoffeeScript (JavaScript). It is being developed by Jan Kuča as the future editor for the web application Školní sešit.cz.

Unlike other similar tools, TakeNote is list-based meaning that the content the editor produces is structured using lists (<ul>, <ol>) and list items (<li>) and one of the key features is its taking advantage of the HTML5 contenteditable attribute. Each of those items has a content node and an optional sub-list:

<ul id="area">
  <li data-type="h1">
    <h1>Lorem ipsum</h1>
  </li>
  <li data-type="paragraph">
    <p>What's up?</p>
    <ul>
      <li data-type="paragraph">
      	<p>Dolor sit amet</p>
      </li>
      <li data-type="paragraph">
      	<p>Dolor sit amet</p>
      </li>
    </ul>
  </li>
  <li data-type="paragraph">
  	<p>Dolor sit amet</p>
  </li>
</ul>
<script type="text/coffeescript" src="./takenote.coffee"></script>
<script type="text/coffeescript">
window.editor = new TakeNote document.getElementById 'area'
editor.activate()
</script>
